What does this program do?
A palindrome is a word that ends and begin the same way: race-car, mom, taco cat, etc. This program takes in an input word and confirms if the word is a palindrome.

Note: install python libraries to run the program.

How to run the program?
1. type the following: 
   python palindrome.py
2. Program will prompt the user to enter a word
3. when the user enters a word, the program tells them if it is a palindrome or not once you press enter
